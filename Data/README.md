# Data Folder

This folder contains the patient-level healthcare dataset used for analysis in this project.

## Dataset Description
- File Name: Patient_Data.xlsx
- Records: 1000 patients
- Level: Individual patient level
- Domain: Healthcare analytics

## Key Fields
- Patient ID
- Age
- Gender
- Department
- Length of Stay
- Total Cost
- Total Revenue

## Notes
- The dataset has been cleaned and validated in Excel.
- This file serves as the single source of truth for all analyses.
